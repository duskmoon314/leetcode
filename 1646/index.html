1646 获取生成数组中的最大值

August 23, 2021 · One min read

可以容易地使用 O(n) 的方法，一维动归算出所有的数，同时记录最大值。当然也可以把计算过程用递归写出来，还可以打表。

C++ Solution​

class Solution {
public:
    int getMaximumGenerated(int n) {
        vector<int> nums = vector<int>(n + 1);
        if (n < 2)
        {
            return n;
        }
        nums[1] = 1;
        int max = 0;
        for (int i = 1; i <= n; ++i)
        {
            int j = i >> 1;
            nums[i] = nums[j] + (i % 2 ? nums[j + 1] : 0);
            max = nums[i] > max ? nums[i] : max;
        }
        return max;
    }
};
